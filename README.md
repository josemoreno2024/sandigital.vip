# 🟩 San Digital — Documentación Técnica Oficial  
**Versión pública del Dossier Maestro v1**

San Digital es un ecosistema digital basado en tecnología blockchain que organiza la participación de usuarios mediante estructuras SAN y una matriz operativa 1x4.  
Este repositorio contiene la documentación técnica oficial del sistema, su arquitectura, trazabilidad y evolución.
